Let's start Visualizing Things!

An open source, intuitive and multi-vendors project to visualize cloud datacenters for Telco Operators

Meet the DC-V project

In Today's datacenter, it's become complex to trace connections, sessions, peering status and application flows especially in multi-vendor environment. This project aims to provide abstraction layer to all of that complexity and help you to find pattern and trace it

Add rich visuals to make complex data clear. Create colorful and interactive links between nodes based on actual running services. Have you always imagined how handy it would be to work with the D3JS framework in Python? Take a look at the online demos.

DC-V includes full HTML, CSS, and custom JavaScript files along with python for easy customization!

Features

Mult-vendors

Designed from beginning to work with multiple datacenter vendors such as Cisco and Juniper fabric

collecting data

Support multiple ways to collect data either directly by accessing the devices or through providing it offline

BEAUTIFUL

Based on D3JS framework that provide visually appealing graphs. Also it support dark mode

Extensible

Support adding new visualization types such as BGP peering types, security flows and BFD sessions

Customizable

just like any other open source project. You are free to download and customize it.

Reporting

Support providing tabular reports based on collected data from nodes
